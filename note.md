App Description
SchemaLence is a PostgreSQL database schema viewer and data exploration tool with a modern desktop GUI. It provides:

Database Connection: Connect via connection string or form fields (host, port, database, user, password)
Schema Browser: Browse all tables in the public schema with a sidebar navigation
Comprehensive Schema Analysis: View table descriptions, columns, foreign keys, primary keys, unique constraints, indexes, triggers, check constraints, ENUM values, and example data
Data Grid: View table data (up to 500 rows) in a sortable treeview
Export & Copy: Export data to CSV, copy schema/data to clipboard
View All Schemas: Generate a complete schema documentation for all tables
Built with Python using tkinter, ttkbootstrap (cyborg theme), psycopg2, and pandas. Packaged as a standalone executable using PyInstaller.
